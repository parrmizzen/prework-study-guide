HTML

  • The Head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The Title element defines the title for the webpage.
  • Metadata is data about our webpage that the browser uses to effectively serve our webpage to users.
  • HTML attribute defines additional information about an element.
  • The Meta element defines metadata such as the character set, description, keywords, author, viewpoint.
  • The Script element embeds client-side scripts in the HTML document.
  • The Link element connects an external resource to the HTML document.
  • The Base element defines the base URL for the webpage.
  • The Style element defines the CSS styles associated with a website.
  • The P Element is used to wrap up most of your text on the page.

CSS

  • Defines how a webpage is presented to users.
  • Internal Style Sheet contains CSS rules for the webpage in the head section of the HTML file.
  • External CSS style sheets is the most common.
  • A selector defines the element or attributes to which rules, or declarations, will apply.
  • Declarations contain two important components:The CSS property we want to apply and the value of the property.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: Checks what branch we are currently working on.
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: to save any changes made to files
  • git commit -m "Write what you changed here.":to commit changes
  • Make sure you are in your own local folder before commiting changes.
  • Git Pull Origin Main: Pulls source from GitHub to make sure we have the most updated commit's and changes.
  • git push origin feature/"featured branch ex." : Pushes your changes to GitHub.

Javascript

  • A variable is named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
